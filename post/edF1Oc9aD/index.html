Mac上用homebrew方式安装vue

· 2021-09-23 · # 编程生涯

1、安装homebrew(如果已经安装过homebrew则跳过)

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

2、通过brew方式安装node和npm

brew install node

3、安装cnpm
基于node.js,利用淘宝npm镜像安装相关依赖。由于国内使用npm会很慢,推荐使用淘宝NPM镜像

npm install -g cnpm --registry=https://registry.npm.taobao.org

4、安装vue-cli

npm install -g vue-cli
